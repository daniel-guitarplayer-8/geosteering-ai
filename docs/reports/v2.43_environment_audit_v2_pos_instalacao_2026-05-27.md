#### Diagnóstico das 4 falhas de `test_simulation_jax_fortran_parity.py`

```python
E   TypeError: compare_fortran_python() got an unexpected keyword argument 'model_name'
```

**Causa raiz**: O commit `5f6a96a feat(O0): Sprint O0 pre-flight — 6 testes
Tier 1 anti-regressão + profile cell` (autor anterior) introduziu chamadas a
`compare_fortran_python(model_name=...)` mas a assinatura real da função
(em `geosteering_ai/simulation/...`) não aceita `model_name`. **Não é
regressão da migração**; é tarefa pendente do Sprint O0.

**Plot twist visual nos tracebacks**: os arquivos `.pyc` em `__pycache__/`
guardam o path original `/Users/daniel/Geosteering_AI/tests/...` (compilado
em macOS antes da migração). O teste é **executado** corretamente do path
Linux, mas o frame no traceback exibe a string macOS. Para limpar:
```bash
find tests -name __pycache__ -type d -exec rm -rf {} +
```
(opcional — cosmético.)
